#  mimiStation

A Vue.js web appication makes music transcribers' lives easier.
(currently only supports Chrome)

URL: https://mimistation.me/

## Project Setup

### With docker-compose

```
git clone https://github.com/tmtakashi/mimiStation.git
cd mimiStation
docker-compose up -d
```

### Without docker

```
git clone https://github.com/tmtakashi/mimiStation.git
cd mimiStation/client
npm install
npm run serve
```
