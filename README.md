## Project Setup

### With docker-compose

```
git clone https://github.com/tmtakashi/mimistation.git
cd mimistation
docker-compose up -d
```

### Without docker

```
git clone https://github.com/tmtakashi/mimistation.git
cd mimistation/client
npm install
npm run serve
```
